Website Access URLs

Local Access (Same Computer)

Use this URL to access the website from this computer:

http://localhost:3000

Local Network Access (Other Devices)

Use this URL to access the website from other devices on the same network:

http://10.1.1.250:3000

Note: This only works for devices on the same network.

External Access (Internet)

To access the website from the internet, you need to use ngrok:

  1. Open a new terminal/command prompt
  2. Run the following command:
npx ngrok http 3000

This will display a public URL that you can use to access your website from anywhere.
